# SiteForYou Theme

**SiteForYou Theme** — це стартова база для кастомної теми WordPress.
Вона містить сучасну структуру з підтримкою Gutenberg-блоків через ACF, PSR-4 автозавантаження класів і Vite для збірки фронтенду.

## Можливості

- PSR-4 автозавантаження класів
- Розбиття Gutenberg-блоків на окремі класи + поля через ACF
- Підключення стилів/скриптів для фронтенду і редактора
- Vite для білду JS та CSS
- Можливість швидко додавати нові блоки і шаблони
